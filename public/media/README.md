# Media drop-folder

Generated background video goes here. The homepage hero looks for:

- `hero-loop.mp4` — H.264, 1080p+, 10–15s seamless loop, **muted**
- `hero-loop.webm` — optional, smaller/sharper for modern browsers

Until these exist, the hero gracefully falls back to the featured film's
backdrop image + animated light shafts (no error).

See `docs/BRAND_ASSETS.md` → section 1 for the exact generation prompt.
